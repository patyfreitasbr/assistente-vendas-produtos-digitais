# Assistente de Vendas com IA para Produtos Digitais

Transforme qualquer conversa em uma oportunidade de venda usando um sistema simples, estratégico e pronto para aplicar.

Este projeto contém um **prompt estruturado** que funciona como um assistente de vendas, ajudando você a entender o cliente, construir ofertas e aumentar suas conversões — mesmo sem experiência em vendas.


## O problema

A maioria das pessoas perde vendas porque:

- Não sabe o que responder no WhatsApp ou direct  
- Oferece produtos sem entender o cliente  
- Trava na conversa e perde oportunidades  
- Não consegue conduzir o atendimento  

👉 Resultado: esforço alto e pouca conversão.


## A solução

Este projeto resolve isso com um sistema que:

- Interpreta o interesse do cliente  
- Classifica o tipo de lead  
- Sugere perguntas estratégicas  
- Indica a melhor oferta  
- Ajuda a aumentar o ticket médio  

👉 Tudo de forma lógica, sem parecer forçado.


## Como funciona

O assistente segue um fluxo de vendas estruturado:

1. Leitura do interesse  
2. Diagnóstico do lead  
3. Perguntas de qualificação  
4. Oferta principal  
5. Oferta complementar (cross-sell)  
6. Estratégia de ancoragem  

👉 Ou seja: você não responde — você conduz a venda.

## Como usar

1. Acesse o arquivo do prompt  
2. Copie o conteúdo  
3. Cole na IA de sua preferência (Gemini, GPT...)
4. Envie o interesse do cliente  

Exemplo:

```

Interesse: Quero começar a ganhar dinheiro na internet

ou

Interesse: Quero crescer no YouTube mas estou travada

```
5. Use a resposta gerada no seu atendimento
